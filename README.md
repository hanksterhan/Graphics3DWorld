# Graphics3DWorld
An interactive 3D world created using WebGL/OpenGL

# Implemented Features 
- Avatar (keys IJKL to move, keys UO to rotate, NM to move closer/farther)
- Sunshine 
- Diffuse
- Spotlight
- Procedural Solid Texturing (buggy)
- Ground Zero (ongoing)
- Pitch Black (ongoing)
- Helicam (upcoming)
- Shining (upcoming)
